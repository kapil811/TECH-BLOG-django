# TECH BLOG – full stack web application| Oct 2023  
##### Developed a  blog web application using Django as backend technology

### ●	Tech used : Django | bootstrap | javascript | jQuery| html |css | tinyMCE 

●	Implimented features : 
- signup  
- login   
- logout
- Djnago messages  
- search    
- comments  
- reply to comment

- views count
